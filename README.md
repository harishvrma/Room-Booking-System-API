# Room Booking API

A RESTful API built using **Node.js, Express, and MySQL** that allows users to register, login, view rooms, and book rooms securely.  
The system also prevents **overlapping bookings** using database transactions.

---

# Tech Stack

- Node.js
- Express.js
- MySQL
- mysql2
- JWT Authentication
- Express Validator
- dotenv
